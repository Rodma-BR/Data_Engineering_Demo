# Steps to access Momentum

To access the gym, a few steps are required. The next figure shows the flow

<img src='../figs/access_flow_momentum_dark.png'></img> 

<ol>
<li> For every new member, they'll need to fill a form and explicitly agreeing the information consent. All data collected is appended to "customers" table.
</li>
<li> Every customer must go to the reception area, they can scan a code if they have paid a plan before or pay for any available plan.
</li>
<li> In case a code is not working or plan suscription has end, then staff will help you either giving you manual access or requesting payment for a new plan, append the information to "payments" table.
</li>
<li> Once payment is validated the access information is appended to "access_registry" table, your climbing journey begins!
</li>
</ol>
